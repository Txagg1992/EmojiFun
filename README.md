This is an API practice/fun app that fetches an Emoji API, and displays a list in a Lazy column. Items are clickable and open a custom dialog. The dialog provides navigation to a detail screen.

API used: https://emoji-api.com/

Tools Used:

  Retrofit, Hilt, 
  Kotlin,
  Coroutines,
  JetPack Compose,
  MVVM/MVI type architecture,
  Navigation with NavGraph and navArgs,
